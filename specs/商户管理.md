# 商户列表

## 搜索区域

按商户名称和代理商两个搜索，代理商搜索框如果输入的是完整的手机号，则使用精确搜索，否则使用模糊搜索（代理商用户名、姓名、手机号、邮箱）

## 列表区域

显示以下字段
- 商户Id
- 商户名称
- 所属代理商姓名（没有显示--）
- 所属代理商手机号（没有显示--）
- 订单总数
- 订单总金额
- 分润总金额（汇总租借费用门店分润金额）
- 操作栏（显示划拨按钮）

列表支持分页查询，默认分页大小10，根据查询参数limit进行控制，小于等于0时自动调整为10。

## 查询逻辑

采用直接写SQL的方式进行联合查询，SQL示例如下
```sql
WITH orders AS(
	SELECT merchant_id,SUM(order_amount) AS order_amount, SUM(merchant_profit) AS merchant_profit FROM merchant_order GROUP BY merchant_id
)

SELECT mch.*, usr.phone,usr.fullname,odr.order_amount,odr.merchant_profit FROM merchant AS mch
LEFT JOIN `user` AS usr ON mch.agent_id = usr.id
LEFT JOIN orders AS odr ON mch.id = odr.merchant_id
```

## 其他

点击划拨按钮弹出窗口，窗口内显示输入框，输入要归属代理商的手机号。点击保存后后端根据手机号查询对应的代理商并将代理商id更新到商户的agent_id字段。
