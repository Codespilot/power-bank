# 登录

1. 登录页面地址：/login。
2. 用户通过输入用户名和密码进行登录，用户名可以是手机号或者邮箱。
3. 连续两次登录失败后页面需要显示一个4位的验证码，用户必须要填写验证码才能登录。
4. API需要检查提交数据来源，如果不是从本站的登录页面提交的数据，不允许操作。
5. 输入的时候需要校验用户名格式，必须是中国大陆手机号或者邮箱。
6. 同一账号每登录失败一次，access_failed_count计数增加1，连续登录超过10后，账号将被锁定，解锁时间记录到locked_out字段，锁定时长为((access_failed_count - 10) * 10)分钟，最长不超过24小时，登录成功后access_failed_count计数重置为0，locked_out清除。
