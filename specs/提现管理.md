# 角色

- 管理员：查询所有提现申请记录，审批提现申请
- 普通用户：仅查询自己提交的申请记录

# 汇总栏（仅管理员显示）

- 已提现总额（审批通过的提现申请金额汇总）
- 待审批总额（待审批状态提现申请金额汇总）
- 带审批数量

# 查询条件

- 用户：用户名、手机号、姓名、邮箱
- 提交日期：根据日期范围查询申请的created_at
- 状态：待审批，已通过、已拒绝

# 列表

显示以下字段：
- 申请时间
- 申请人（姓名+手机号）
- 申请金额
- 当前可用
- 冻结金额
- 总金额
- 审批状态
- 审批人（姓名）
- 审批意见
- 操作（同意、拒绝、作废，作废按钮仅显示在当前用户数据后面）

# 审批

- 同意：直接修改状态为通过，审批意见修改为同意，同时记录对应用户钱包流水，流水发生时间为当前时间，备注为提现审批通过
- 拒绝：弹出窗口填写拒绝原因，窗口标题“审批意见”，提交后仅更新状态和审批意见、审批人。
- 作废：仅限数据归属用户操作，点击后弹窗确认。
