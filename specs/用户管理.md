
# 用户列表

## 操作区域
1. 关键字输入框，输入后按回车键搜索用户名、手机号、影响、姓名中包含关键字的用户。
2. 显示新增按钮，点击后弹出新增用户的dialog，输入项包括用户名、姓名、手机号、邮箱、密码，点击保存按钮提交，用户新增完成后，以用户id作为主键新增一条用户资产数据。
3. 用户名仅能包含字母、数字和下划线且必须包含字母，长度不超过32，手机号仅支持中国大陆手机号码，除邮箱外其他必填。
4. 新建用户时需要校验用户名、手机号、邮箱（如果有）的唯一性和格式。

## 列表区域

1. 显示用户名，手机号、邮箱、注册时间（创建时间）、总资产、锁定解除时间、操作等列。
2. 操作列显示查看和重置密码按钮。
3. 底部显示分页按钮，分页大小固定位20条。
4. 点击操作列的查看按钮弹出用户详情dialog，显示用户名、姓名、手机号、邮箱、资产情况。
5. 用户列表查询时默认按照创建时间倒序排列。
