# 功能描述

用于管理用户自己的银行卡信息

# 对应表

表结构.md --> 银行卡

# API接口

- 列表查询：GET /api/bankcards?keyword=xxx&limit=10&page=1
- 获取详情：GET /api/bankcards/{id} 非管理员需要检查user_id是否等于当前用户id
- 新增：POST /api/bankcards
- 更新：POST /api/bankcards/{id}
- 删除：DELETE /api/bankcards/{id}
- 下拉列表 GET /api/bankcards/lookup 及返回id，card_no，name三个字段

# 列表

## 操作区域

关键字（keyword）：搜索银行卡号、身份证号、持卡人姓名、预留手机号

新增：跳转到新增页面

## 列表

- 卡号
- 银行卡照片（显示点击查看，点击预览大图，图片URL：card_photo_url）
- 持卡人姓名
- 持卡人身份证号
- 预留手机号
- 身份证照片（显示国徽面和人像面，点击预览大图）
- 操作（查看、编辑、删除）

非管理员仅查询当前用户数据，分页查询，默认每页10条

响应数据结构
```json
{
    "id": 12342, // id
    "card_no":"6222", //银行卡号
    "name": "张三", //持卡人姓名
    "id_no": "10001", //持卡人身份证号，隐藏中间6位
    "mobile": "133xxx", //预留手机号
    "card_photo_url": "https://xxx.com/files/attachments/xxxx.png?token=xxx", //银行卡照片URL
    "id_photo_badge_url": "https://xxx.com/files/attachments/xxxx.png?token=xxx", //身份证国徽照片URL
    "id_photo_face_url": "https://xxx.com/files/attachments/xxxx.png?token=xxx", //身份证人像照片URL
    "is_default":true, //是否默认
    "created_at": "yyyy-MM-dd HH:mm:ss"
}
```

# 新增和编辑

- 身份证（上传身份证国徽面和人像面照片）
- 持卡人姓名
- 持卡人身份证
- 银行卡号
- 银行卡照片
- 预留手机号
- 是否默认

银行卡号不能重复，上传接口调用`/api/attachments/upload/form`

请求参数
```json
{
    "card_no":"6222", //银行卡号
    "name": "张三", //持卡人姓名
    "id_no": "10001", //持卡人身份证号，隐藏中间6位
    "mobile": "133xxx", //预留手机号
    "card_photo": "文件名.png", //银行卡照片，上传接口返回的file_name
    "id_photo_badge": "文件名.png", //身份证国徽照片，上传接口返回的file_name
    "id_photo_face": "文件名.png", //身份证人像照片，上传接口返回的file_name
    "is_default":true, //是否默认
}
```
